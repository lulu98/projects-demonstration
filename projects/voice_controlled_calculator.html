Project: Voice Controlled Calculator

Description

A calculator that can be controlled by either pressing the buttons or by using your voice. The application uses the Speech Recognition package from Microsoft (Zira). The mechanism is impelmented as follows:

  1. Speech Recognition translates spoken words into a string for further processing
  2. Infix Notation to Postfix Notation
  3. Evaluation of Postfix Expression

Tags: C#, .NET, Visual Studio, Speech Recognition
