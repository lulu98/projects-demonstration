Project: Retro Games

Pong

Description

A web application featuring Pong. The player can decide if he either wants to play against another person or against the computer. The ball is simulated with a simple formula that translates the incidence angle into the emergent angle when hitting the walls. The computer is implemented by predicting the future position of the ball and moving towards this destination.

Tags: HTML5, CSS, JavaScript, PHP, Pixel Animation, Web Development

Snake

Description

A web application featuring Snake. The player can control the snake by using the arrow keys on the keyboard. He needs to collect the apples. When eating an apple, the length of the snake is incremented by one. If the snake's head is bumping into its body, the player has lost. Afterwards, the player is asked to enter his name and will be redirected to the highscore page.

Tags: HTML5, CSS, JavaScript, PHP, Pixel Animation, Web Development

GitHub | Snake

Tetris

Description

A web application featuring Tetris. The player can control the shapes by using the arrow keys on the keyboard. In the background the original Tetris soundtrack is playing. The player can pause or reset the game whenever he likes.

Space Invaders

Description
